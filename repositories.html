Repositories

All software developed by the Digital C-Fiber group, organized by research focus.

FAIR Microneurography

openMNGlab
Python

A Python framework for microneurography data analysis. Provides a unified interface for loading, processing, and analyzing electrophysiology recordings from MNG experiments.

framework MNG
PyDapsys
Python

Open-source Python library for reading proprietary DAPSYS electrophysiology files. Reverse-engineered format reader that exports to NIX for long-term, interoperable data access.

data access DAPSYS NIX
odMLtablesForMNG
Python

odML-Tables adapted as a metadata standard for microneurography. Provides templates and tooling for structured metadata management in MNG lab workflows.

metadata FAIR odML
odmltables-for-ldh
Python

Extension of odML-Tables to align microneurography metadata with the FAIRDOM-SEEK Local Data Hub schema. Enables semi-automatic upload and anonymization for NFDI4Health data sharing.

NFDI4Health LDH FAIR
python-odmltables
Python

Interface to convert odML structures to and from table-like representations such as spreadsheets. Core library underpinning the MNG metadata toolchain.

odML metadata

Simulational Modelling of Peripheral Nerves

NEURON-C-fiber
NEURON

Biophysical simulation of an unmyelinated C-fiber axon using the NEURON simulation environment. Models activity-dependent slowing and axonal excitability via Hodgkin-Huxley equations.

HH model ADS
DRG-soma-model
Model

Model of the dorsal root ganglion (DRG) soma. Complements the C-fiber axon model for whole-neuron simulation of peripheral nociceptors.

DRG simulation
Modelling-of-Memory-in-Unmyelinated-Axons
Python

Efficient model predicting conduction velocity changes in unmyelinated axons via a memory function — incorporating linear long-term and non-linear short-term memory to capture how prior activity modulates AP speeds.

ADS memory model

Microneurography Data Analysis

SpikeSortingPipeline
Notebook

Jupyter notebook pipeline for spike detection and sorting by morphology in single-electrode MNG recordings. Implements the supervised approach from Troglio et al. 2024.

spike sorting SVM
SpikeSortingForSpikingPatterns
Python

Spike sorting approach adapted for the analysis of spiking patterns — complementing the core pipeline with pattern-aware classification.

spike sorting patterns
multi-task-learning-for-spike-sorting
Python

Multi-task machine learning for spike sorting in MNG. Learns the core sorting task across recordings with different fibers and spike morphologies — enabling cross-recording generalization without recalibration.

multi-task ML deep learning
SpikeSpy
Python

GUI tool for annotating stimulus-locked electrophysiology data from microneurography experiments. Supports the ground-truth labelling workflow used in sorting validation.

GUI annotation
Spikelets
Python

Analysis tooling for spikelet detection and characterization in peripheral nerve recordings.

detection
