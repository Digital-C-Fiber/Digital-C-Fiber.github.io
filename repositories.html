Repositories

Open-source tools, pipelines, and data standards developed by the Digital C-Fiber group.

openMNGlab

Open-source data analysis framework for microneurography. Loads data from Spike2 and Dapsys, provides visualization, fiber tracking, and feature extraction for machine learning.

Python Data Analysis
spike-sorting-pipeline

Supervised spike sorting pipeline for C-fiber microneurography. Combines knowledge-driven template extraction with data-driven classification (SVM, XGBoost) for improved F1-scores over Spike2.

Python Data Analysis
MNG-CV-modelling

Polynomial regression models for predicting activity-dependent conduction velocity fluctuations in peripheral C-fibers. Used for spike sorting enhancement.

Python Modelling
odml-tables-MNG

odML-based metadata templates and GUI extension for microneurography. Enables structured metadata capture, database search, and semi-automatic upload to NFDI4Health Local Data Hubs.

Python FAIR & Metadata
explainable_MNG_spike_detection

Neural network spike detector using Weighted Hermite Variable Projection Neural Networks. Explainable AI approach for C-fiber MNG data, with sliding window classification and post-hoc analysis.

Python Data Analysis
